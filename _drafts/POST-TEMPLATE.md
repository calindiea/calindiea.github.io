## Best Practices Checklist

Before publishing, make sure you've:

- [ ] Written a compelling title (50-60 characters)
- [ ] Added a concise excerpt (under 160 characters)
- [ ] Written a meta description (150-160 characters)
- [ ] Selected relevant categories (1-3)
- [ ] Added 3-5 relevant tags
- [ ] Included a featured image (1200x630px recommended)
- [ ] Added alt text to all images
- [ ] Used proper heading hierarchy (H1 → H2 → H3)
- [ ] Included relevant internal links to other posts
- [ ] Proofread for spelling and grammar
- [ ] Verified all links work correctly
- [ ] Added keywords naturally throughout content

## SEO Tips

1. **Use keywords naturally** in your title, first paragraph, headings, and throughout
2. **Write for humans first**, then optimize for search engines
3. **Use descriptive headings** that tell readers what each section covers
4. **Add internal links** to related posts/pages on your site
5. **Include images** with descriptive filenames and alt text
6. **Write comprehensive content** - longer, valuable posts tend to rank better
7. **Make it readable** - use short paragraphs, bullet points, and formatting
